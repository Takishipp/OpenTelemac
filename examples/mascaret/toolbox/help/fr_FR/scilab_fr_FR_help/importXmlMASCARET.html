importXmlMASCARET

Fonction d'importation d'un modèle ou d'un état MASCARET au format XML

Séquence d'appel

[erreur] = importXmlMASCARET(id,nomFichier,importModele)

Arguments

erreur

Argument de sortie indiquant la présence d'une erreur si différent de zéro

id

Identifiant de l'instance MASCARET retourné par createMASCARET

nomFichier

Nom du fichier XML contenant le modèle ou l'état MASCARET. La taille de cette chaîne de doit pas dépasser 255 caractères.

importModele

Si vrai (valeur 1), importation du modèle, sinon importation de l'état.

Description

Fonction d'importation d'un modèle ou d'un état MASCARET au format XML.

Cette fonction permet d'importer un fichier XML contenant le modèle ou l'état d'une instance de MASCARET produit par la fonction exportXmlMASCARET

Avant d'utiliser cette fonction, il est nécessaire d'avoir initialisé une instance d'un modèle MASCARET, c'est-à-dire d'avoir fait appel à la fonction createMASCARET. Si l'on souhaite importer l'état, il faut avoir initialisé le modèle, c'est-à-dire avoir fait appel à une des fonctions importModelMASCARET ou importXmlMASCARET avec le dernier parametre à 1.

Exemple

toolbox_dir=getenv("toolbox_dir");
c = filesep();        
    
// creation of the MASCARET model
[err, id] = createMASCARET();
assert_checktrue(id>0);

// import Model and State
err = importXmlMASCARET(id,"file:"+toolbox_dir+c+"demos"+c+"Help"+c+"demo1"+c+"demo1_Model.xml",1);
assert_checkequal(err,0);    
err = importXmlMASCARET(id,"file:"+toolbox_dir+c+"demos"+c+"Help"+c+"demo1"+c+"demo1_State.xml",0);
assert_checkequal(err,0);    

// model deletion
err = deleteMASCARET(id);
assert_checkequal(err,0);
