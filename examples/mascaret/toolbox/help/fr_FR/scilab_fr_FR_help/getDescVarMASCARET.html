getDescVarMASCARET

Récupère la liste des variables de MASCARET accompagnée d'une description

Séquence d'appel

[erreur,TabVar,TabDesc]=getDescVarMASCARET(id)

Arguments

erreur

Argument de sortie indiquant la présence d'erreur si différent de zero

TabVar

Tableau des noms de variable du modèle ou de l'état

TabDesc

Tableau des descriptions de variable du modèle ou de l'état

id

Identifiant de l'instance MASCARET retourné par createMASCARET

Description

Récupère la liste complète des variables de MASCARET accompagnée d'une description

Exemple

// creation of the MASCARET model
[err, id] = createMASCARET();
assert_checktrue(id>0);

// get the description
[err,TabVar,TabDesc]=getDescVarMASCARET(id);
assert_checkequal(err,0);
disp(TabVar+" : "+TabDesc);

// model deletion
err = deleteMASCARET(id);
assert_checkequal(err,0);
