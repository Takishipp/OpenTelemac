getVersionMASCARET

Renvoie le numéro de version du noyau de calcul MASCARET

Séquence d'appel

[a,b,c]=getVersionMASCARET()

Arguments

a

Identifiant de la version majeur de MASCARET

b

Identifiant de la version mineure de MASCARET

c

Identifiant de la version micro de MASCARET

Description

Renvoie le numéro de version du noyau de calcul MASCARET au format a.b.c

Exemple

toolbox_dir=getenv("toolbox_dir");
c = filesep();

// version of the MASCARET computation kernel
[a,b,c] = getVersionMASCARET();

disp(strcat([ascii(a+48) '.' ascii(b+48) '.' ascii(c+48)]),'MASCARET version : ');
