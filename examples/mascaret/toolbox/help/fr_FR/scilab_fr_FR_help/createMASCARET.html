createMASCARET

Initialise les ressources associées à une instance de MASCARET

Séquence d'appel

[erreur, id] = createMASCARET()

Arguments

erreur

Argument de sortie indiquant la présence d'erreur si différent de zero

id

Identifiant de l'instance MASCARET utilisé en argument des autres fonctions de l'API

Description

Création d'une instance de MASCARET.

Cette fonction permet d'initialiser les ressources (allocation mémoire et initialisation de variables) associées à une instance MASCARET, c'est-à-dire les ressources concernant un modèle et son état.

C'est cette fonction qui doit être utilisée en premier avant de mettre en oeuvre un modèle et lancer des simulations. Il n'y a aucun pré-requis avant de pouvoir utiliser cette fonction.

Exemple

// creation of 3 MASCARET models
id = zeros(3,1);
for i = 1:3
   [err, id(i)] = createMASCARET();
   assert_checkequal(err,0);  
   printf("\tA new model is created with the identification number :%d\n", id(i));
end

// deletions
for i = 1:3
   err = deleteMASCARET(id(i));
   assert_checkequal(err,0);  
end
