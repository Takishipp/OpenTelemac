importXmlMASCARET

Export the MASCARET model or state in a file with the XML format

Calling sequence

[error] = importXmlMASCARET(id,FileName,importModel)

Arguments

error

(Output argument) Error code value : zero indicates that no errors occurred

id

MASCARET instance number returned by the function createMASCARET

FileName

Name of the XML file containing the MASCARET model or state. The length of this string does not exceed 255 characters.

importModel

If true (value = 1), import a model. If not true (value = 0), import a state.

Description

Export the MASCARET model or state in a file with the XML format

This function gives the possibility to import XML files containing a MASCARET model or state previously created by the function exportXmlMASCARET

Before using this function, it is necessary to create a MASCARET instance, with the help of the function createMASCARET. If the state is imported, the model has to be initialized, with the help of one of the functions importModelMASCARET or importXmlMASCARET with the last parameter equal to 1.

Example

toolbox_dir=getenv("toolbox_dir");
c = filesep();        
    
// creation of the MASCARET model
[err, id] = createMASCARET();
assert_checktrue(id>0);

// import Model and State
err = importXmlMASCARET(id,"file:"+toolbox_dir+c+"demos"+c+"Help"+c+"demo1"+c+"demo1_Model.xml",1);
assert_checkequal(err,0);    
err = importXmlMASCARET(id,"file:"+toolbox_dir+c+"demos"+c+"Help"+c+"demo1"+c+"demo1_State.xml",0);
assert_checkequal(err,0);    

// model deletion
err = deleteMASCARET(id);
assert_checkequal(err,0);
