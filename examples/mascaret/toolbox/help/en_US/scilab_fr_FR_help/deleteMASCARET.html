deleteMASCARET

Free all the variables of MASCARET for a specific instance

Calling sequence

error = deleteMASCARET(id)

Arguments

id

Identification number of the MASCARET instance returned by createMASCARET

error

(Output argument) Error code value : zero indicates that no errors occurred

Description

Free all the variables of MASCARET for a specific instance

Example

// creation of 10 MASCARET models
id = zeros(10,1);
for i = 1:10
   [err, id(i)] = createMASCARET();
   assert_checkequal(err,0);  
   printf("\tA new model is created with the identification number :%d\n", id(i));
end

// deletions
for i = 1:10
   err = deleteMASCARET(id(i));
   assert_checkequal(err,0);  
   printf("\tThe model is destroyed :%d\n", id(i));
end
