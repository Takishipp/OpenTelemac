getDescVarMASCARET

Gives the list of all the MASCARET variables with a description

Calling sequence

[error,TabVar,TabDesc]=getDescVarMASCARET(id)

Arguments

error

(Output argument) Error code value : zero indicates that no errors occurred

TabVar

Table variable names for the model and for the state

TabDesc

Table variable description for the model and for the state

id

Identification number of the MASCARET instance returned by createMASCARET

Description

Gives the list of all the MASCARET variables with a description

Example

// creation of the MASCARET model
[err, id] = createMASCARET();
assert_checktrue(id>0);

// get the description
[err,TabVar,TabDesc]=getDescVarMASCARET(id);
assert_checkequal(err,0);
disp(TabVar+" : "+TabDesc);

// model deletion
err = deleteMASCARET(id);
assert_checkequal(err,0);
