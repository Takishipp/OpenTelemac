readme.txt of the MASCARET SCILAB toolbox

This toolbox uses MASCARET with the API.

In order to use this toolbox you have to :
(1) Compile the MASCARET dynamic library using the provided Makefile in the directory '../src'
(2) Launch SCILAB v5.4.1 or above
(3) Run the scripts in SCILAB 'builder.sce' followed by 'loader.sce'

SCILAB wrappers for the MASCARET API are macros. See the on-line help for using them.

MASCARET have several test cases that can be run using the SCILAB command '-->test_run(".")' in the toolbox directory.
